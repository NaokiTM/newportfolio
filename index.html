knights tour solver
A program to visually solve the famous Knights Tour problem, showing possible moves from the knights position and highlighting previous moves. it implements the Warnsdorffs heuristic to look ahead and greedily choose the move with the largest number of possible outgoing moves from it. Pygame provides a simple and customizable user interface, and in the future I plan to add a mode for the user to be able to solve it themselves as well as solving for custom board sizes
